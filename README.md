# libgamestudio

A library that tries to implement the file formats utilized by Gamestudio / Acknex Engine:

- `WMB7`
- `MDL3`, `MDL4`, `MDL5`, `MDL7`
- `HMP5`

## Status

- [x] `WMB7`
- [x] `MDL3`
- [ ] `MDL4` (test file missing, should work in theory)
- [x] `MDL5`
- [ ] `MDL7`
- [ ] `HMP5`
